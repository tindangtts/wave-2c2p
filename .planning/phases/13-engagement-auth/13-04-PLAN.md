**Step D — `src/app/api/auth/webauthn/register/verify/route.ts`** (verifies registration + stores credential):

Import: `verifyRegistrationResponse` from `@simplewebauthn/server`, createClient, NextResponse, isDemoMode.

POST handler accepts `{ credential }` JSON body:
- Demo mode: mark biometric enrolled in user_profiles: `.update({ webauthn_credential_id: 'demo-credential', webauthn_public_key: 'demo-key', webauthn_counter: 0 }).eq('id', DEMO_USER_ID)`. Return `{ enrolled: true }`.
- Real: authenticate user, fetch `webauthn_challenge` from user_profiles.
- Call `verifyRegistrationResponse({ response: credential, expectedChallenge: profile.webauthn_challenge, expectedOrigin: process.env.NEXT_PUBLIC_APP_ORIGIN ?? 'http://localhost:3000', expectedRPID: process.env.NEXT_PUBLIC_DOMAIN ?? 'localhost', requireUserVerification: true })`.
- If `verified`: store `registrationInfo.credential.id` as `webauthn_credential_id`, base64url-encode `registrationInfo.credential.publicKey` as `webauthn_public_key`, store `registrationInfo.credential.counter` as `webauthn_counter`.
- Clear `webauthn_challenge` from profile (set null).
- Return `{ enrolled: true }` or `{ error: 'Verification failed' }` 400.

4. Add `handleBiometricToggle` function:
   ```typescript
   async function handleBiometricToggle(enabled: boolean) {
     if (!enabled) {
       // Disable: clear credential from user_profiles
       const supabase = createClient()
       await supabase.from('user_profiles').update({
         webauthn_credential_id: null,
         webauthn_public_key: null,
         webauthn_counter: 0,
       }).eq('id', (await supabase.auth.getUser()).data.user!.id)
       setBiometricsEnabled(false)
       return
     }
     // Enable: check availability first
     const available = await PublicKeyCredential.isUserVerifyingPlatformAuthenticatorAvailable().catch(() => false)
     if (!available) {
       toast.error(t('menu.biometricsUnavailable'))
       return
     }
     try {
       // Get registration options from server
       const optRes = await fetch('/api/auth/webauthn/register', { method: 'POST' })
       const { options, mock } = await optRes.json()
       let credential = null
       if (!mock) {
         credential = await startRegistration({ optionsJSON: options })
       }
       // Verify on server
       const verifyRes = await fetch('/api/auth/webauthn/register/verify', {
         method: 'POST',
         headers: { 'Content-Type': 'application/json' },
         body: JSON.stringify({ credential }),
       })
       const { enrolled } = await verifyRes.json()
       if (enrolled) {
         setBiometricsEnabled(true)
         toast.success(t('menu.biometricsEnrolled'))
       } else {
         toast.error(t('menu.biometricsEnrollFailed'))
       }
     } catch {
       toast.error(t('menu.biometricsEnrollFailed'))
     }
   }
   ```
5. Add `import { toast } from 'sonner'` if not already imported.
