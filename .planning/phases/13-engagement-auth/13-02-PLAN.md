3. **GET handler**: 
   - Demo: return `{ notifications: DEMO_NOTIFICATIONS }`.
   - Real: authenticate user, query `supabase.from('notifications').select('*').eq('user_id', user.id).order('created_at', { ascending: false }).limit(50)`. Return `{ notifications: data ?? [] }`.

4. **PATCH handler** (accepts JSON body):
   - Parse body: `{ id?: string, all?: boolean }`.
   - Demo: return `{ success: true }` (no-op).
   - Real: authenticate user.
     - If `all === true`: `.update({ is_read: true }).eq('user_id', user.id).eq('is_read', false)`.
     - If `id`: `.update({ is_read: true }).eq('id', id).eq('user_id', user.id)`.
     - Else 400 `{ error: 'Provide id or all:true' }`.
   - Return `{ success: true }`.
5. Wrap both in try/catch, return 500 on error.
