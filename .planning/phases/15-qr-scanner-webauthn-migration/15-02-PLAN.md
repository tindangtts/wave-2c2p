  | { type: 'p2p_wallet'; walletId: string }

3. `detectQRType(rawValue: string): QRType` — pure function:
   - Trim the input
   - If `P2P_WALLET_REGEX.test(trimmed)` → return `{ type: 'p2p_wallet', walletId: trimmed }`
   - Else → return `{ type: 'unknown', rawValue: trimmed }`

4. `decodeQRFromFile(file: File): Promise<string | null>` — async function:
   - Check if `BarcodeDetector` exists in window. If not, dynamically import the polyfill:
     ```typescript
     if (typeof window !== 'undefined' && !('BarcodeDetector' in window)) {
       const { BarcodeDetector: BDPoly } = await import('barcode-detector/pure')
       // @ts-expect-error -- polyfill global assignment
       window.BarcodeDetector = BDPoly
     }
     ```
   - Create a BarcodeDetector instance: `new BarcodeDetector({ formats: ['qr_code'] })`
   - Create ImageBitmap from file: `const bitmap = await createImageBitmap(file)`
   - Detect: `const results = await detector.detect(bitmap)`
   - Return `results[0]?.rawValue ?? null`
   - Wrap in try/catch, return `null` on any error
   - Add `// @ts-expect-error -- BarcodeDetector not in TS lib types` where needed for the `new BarcodeDetector()` call

IMPORTANT: Do NOT import barcode-detector at top level — it must be a dynamic import to avoid SSR issues. The `BarcodeDetector` type is not in TypeScript's lib.dom.d.ts, so use `@ts-expect-error` annotations.
