  <action>
1. Install jsPDF: `npm install jspdf` (jsPDF 4.x bundles autotable — no separate package needed)

2. Create `src/app/api/statement/route.ts` — a GET endpoint that returns ALL transactions for a date range without pagination. Follow the same auth + demo mode pattern as `/api/notifications/route.ts`.
   - Query params: `dateFrom` (required, yyyy-MM-dd), `dateTo` (required, yyyy-MM-dd)
   - Demo mode: filter DEMO_TRANSACTIONS by date range, return `{ transactions: Transaction[] }`
   - Auth mode: query Supabase `transactions` table with `.eq('user_id', user.id).gte('created_at', dateFrom).lte('created_at', dateTo + 'T23:59:59').order('created_at', { ascending: false }).limit(500)`
   - Return 400 if dateFrom or dateTo missing
   - Return 401 if not authenticated (non-demo)
   - Return `{ transactions: Transaction[] }`

3. Create `src/lib/pdf/generate-statement.ts` with a single exported async function:
   ```typescript
   export async function generateStatement(
     transactions: Transaction[],
     dateFrom: string,
     dateTo: string
   ): Promise<void>
   ```
   - Dynamic import jsPDF: `const { jsPDF } = await import('jspdf')` then `await import('jspdf-autotable')` for the side-effect registration
   - Create A4 portrait document: `new jsPDF({ orientation: 'portrait', unit: 'mm', format: 'a4' })`
   - Header: "2C2P Wave - Transaction Statement" at (14, 18), font size 14
   - Subheader: "Period: {dateFrom} to {dateTo}" at (14, 26), font size 9
   - Generated line: "Generated: {format(new Date(), 'dd MMM yyyy HH:mm')}" at (14, 31), font size 9 — use `format` from `date-fns`
   - Table using `(doc as any).autoTable({...})`:
     - startY: 38
     - head: [['Date', 'Description', 'Type', 'Amount', 'Status']]
     - body: map transactions to rows: [format date as 'dd MMM yyyy', tx.description, tx.type.replace('_', ' '), formatCurrency(tx.amount, tx.currency as CurrencyCode), tx.status]
     - styles: { fontSize: 8, cellPadding: 2 }
     - headStyles: { fillColor: [255, 230, 0], textColor: [33, 33, 33] } — brand yellow header
   - Footer: "Total transactions: {transactions.length}" below the table
   - Save: `doc.save(\`statement-${dateFrom}-${dateTo}.pdf\`)`
   - All labels in English only (no custom font embedding needed — per research recommendation)
  </action>
