   ```typescript

     standard: { daily_limit_satang: 3000000,  monthly_limit_satang: 10000000 },  // 30,000 / 100,000 THB
   ```

     a. Get user via `supabase.auth.getUser()` — return 401 if no user
     b. Query `user_profiles` for `daily_limit_satang, monthly_limit_satang` where id = user.id
     c. Return `{ daily_limit_satang: data?.daily_limit_satang ?? 5000000, monthly_limit_satang: data?.monthly_limit_satang ?? 20000000 }`
   - Note: columns may not exist yet in deployed DB — the `?? defaultValue` fallback handles this gracefully

   **Before creating the API route**, append the spending limits migration SQL to `.planning/supabase-schema.sql`:

   ```sql
     ADD COLUMN IF NOT EXISTS monthly_limit_satang bigint DEFAULT 20000000;
   ```
   This must be run in Supabase Dashboard SQL Editor before PATCH will persist in non-demo mode.

   **PATCH handler:**
   - Parse body with Zod schema:

   - Validate body — return 400 with `{ error: 'Invalid tier' }` on failure
   - Demo mode: return `NextResponse.json({ success: true })`
   - Auth mode:
     a. Get user — return 401 if no user
     b. Look up tier values from SPENDING_TIERS
     c. Update `user_profiles` set `daily_limit_satang` and `monthly_limit_satang` where id = user.id
     d. Return `NextResponse.json({ success: true, ...tierValues })`
   - Wrap everything in try/catch, return 500 on internal error

2. Create `src/hooks/use-spending-limits.ts`:
   ```typescript
   'use client'
   import useSWR from 'swr'

   interface SpendingLimits {
     daily_limit_satang: number
     monthly_limit_satang: number
   }

   const fetcher = (url: string): Promise<SpendingLimits> =>
     fetch(url).then((r) => {
       if (!r.ok) throw new Error('Failed to fetch spending limits')
       return r.json()
     })

   export function useSpendingLimits() {
     const { data, isLoading, error, mutate } = useSWR<SpendingLimits>(
       '/api/spending-limits',
       fetcher
     )
     return {
       dailyLimitSatang: data?.daily_limit_satang ?? 5000000,
       monthlyLimitSatang: data?.monthly_limit_satang ?? 20000000,
       isLoading,
       error,
       mutate,
     }
   }
   ```
