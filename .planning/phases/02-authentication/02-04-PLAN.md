    2. Create `src/app/api/auth/passcode/setup/route.ts`:
       - POST receives `{ passcode }` (6-digit string)
       - Validate: must be exactly 6 digits (regex /^\d{6}$/)
       - Get authenticated user via supabase.auth.getUser() — return 401 if none
       - Import hashPasscode from src/lib/auth/passcode.ts
       - Hash the passcode: `const hash = hashPasscode(passcode)`
       - Update user_profiles atomically: SET passcode_hash = hash, registration_complete = true, registration_step = 3, passcode_attempts = 0
       - Return `{ success: true }`
       - This is the FINAL registration step — registration_complete=true triggers redirect to /home on next login

    3. Create `src/app/api/auth/passcode/verify/route.ts`:
       - POST receives `{ passcode }` (6-digit string)
       - Get authenticated user via supabase.auth.getUser() — return 401 if none
       - Fetch user_profiles row: SELECT passcode_hash, passcode_attempts, passcode_locked_at
       - If passcode_locked_at is set and less than 30 minutes ago: return 423 `{ error: 'Account locked', locked: true }`
       - If passcode_locked_at is set and more than 30 minutes ago: reset lock (UPDATE passcode_attempts=0, passcode_locked_at=null)
       - Import verifyPasscode from src/lib/auth/passcode.ts
       - If verifyPasscode returns true: UPDATE passcode_attempts=0, return `{ success: true }`
       - If verifyPasscode returns false:
         - Increment passcode_attempts: UPDATE passcode_attempts = passcode_attempts + 1
         - If new attempts >= 5: UPDATE passcode_locked_at = now()
         - Return 401 `{ error: 'Incorrect passcode', remaining: 5 - newAttempts, locked: newAttempts >= 5 }`
  </action>
