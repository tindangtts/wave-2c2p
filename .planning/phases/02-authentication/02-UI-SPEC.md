- Welcome screen subtitle: `--text-body` at weight 400, color `#757575` — "Sign in or create an account to get started"

- Registration section subtitles (e.g., "Tell us about yourself"): `--text-body` at weight 400, color `#757575`
- Step label text (e.g., "Step 2 of 3"): `--text-caption` (12px, 400, uppercase, letter-spacing 0.08em)

| Property | Value |

## Interaction Contracts

### OTP Auto-Submit

- Trigger: 6th digit entered via `InputOTP` component
- Action: call verify API immediately, set button to loading state
- Visual: button shows `Loader2` spinner + "Verifying..." text (prevents double-submit)
- On success: navigate per redirect logic
- On error: shake animation on OTP group, clear all slots, refocus slot 1, show error message

### Passcode Auto-Submit

- Trigger: 6th dot fills
- Action: call verify API immediately
- Visual: dot row shows brief "pulse" animation while loading
- On error: shake animation, clear all dots

### Registration Persistence (D-06)

- Zustand store key: `wave-registration-state`
- localStorage sync: `zustand/middleware` `persist` middleware
- Persisted fields: `{ step, firstName, lastName, dateOfBirth, nationality, idType, idNumber, idExpiry, phone }`
- Clear on: successful registration completion
- Resume: on app load, if `registration_complete=false` in `user_profiles`, read Zustand store and redirect to correct step route

### Form Validation Timing

- Per D-07 in CONTEXT.md: validate on "Next" tap (not on-blur)
- Exception: phone format validation shows on blur after first interaction
- Error display: inline below field, `role="alert"` for screen reader announcement
- `aria-invalid="true"` set on field when error is present

### Loading States

| Action | Loading indicator |
|--------|-----------------|
| Send OTP request | CTA button: `Loader2` spinner + "Sending OTP..." |
| Verify OTP | CTA button: `Loader2` spinner + "Verifying..." |
| Save step data | CTA button: `Loader2` spinner + "Saving..." |
| Create passcode | Dot row pulse, no CTA (auto-submit) |
| Passcode verify | Dot row pulse animation |

`Loader2` icon: `lucide-react`, 16px, `animate-spin`, placed left of label text inside button.
