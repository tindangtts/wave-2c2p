## Typography

Inherited from Phase 1 Foundation — no new sizes or weights introduced.

| Role | Size | Weight | Line Height | Letter Spacing | Usage in Phase 7 |
|------|------|--------|-------------|----------------|------------------|
| Display | 28px / 1.75rem | 700 (bold) | 1.2 | -0.02em | Not used |
| Heading | 20px / 1.25rem | 700 (bold) | 1.3 | -0.01em | Screen titles in yellow headers ("Profile Setting", "Visa Card", "Refer Friends", "Notification Settings"); card number revealed state |
| Body | 16px / 1rem | 400 (regular) | 1.5 | 0 | ProfileMenuItem labels; user name in ProfileHeader; language option labels; card holder name on card face; notification toggle labels |
| Caption | 12px / 0.75rem | 400 (regular) | 1.4 | 0.01em | ProfileMenuSection headings ("Biometrics", "Settings", "Help & Support"); language badge next to "Language" item; "V 1.0" version text; referral monthly count; card expiry label |

Two weights only: 700 (bold) for headings and emphasis, 400 (regular) for body and caption.

**Phase 7-specific typography rules:**

- Profile avatar initials: 24px weight 700 `#212121` on `#FFE600` circle — two-letter initials (e.g. "LT")
- User full name in ProfileHeader: 16px weight 700 `#212121` — displayed below avatar
- ProfileMenuSection heading: 12px weight 400 `#9E9E9E` — all-caps ("BIOMETRICS", "SETTINGS", "HELP & SUPPORT") — tracks uppercase via CSS `text-transform: uppercase; letter-spacing: 0.08em`
- ProfileMenuItem label: 16px weight 400 `#212121`
- ProfileMenuItem trailing badge (language): 12px weight 400 `#0091EA` on `#E3F2FD` pill — "EN" / "TH" / "MM"
- Logout button label: 16px weight 700 `#F44336` — destructive red on outlined button
- Card face — masked number: 20px weight 700 `#FFFFFF` — "•••• •••• •••• 1234"; revealed number uses same style
- Card face — holder name: 12px weight 400 `#FFFFFF` — uppercase via CSS `text-transform: uppercase`
- Card face — expiry label: 10px weight 400 `rgba(255,255,255,0.7)` — "VALID THRU"
- Card face — expiry value: 12px weight 700 `#FFFFFF` — "12/28"
- Card face — network logo: VISA wordmark SVG 32px wide
- "Card Frozen" overlay text: 16px weight 700 `#FFFFFF` — centered on card freeze overlay
- Referral code display: 24px weight 700 `#212121` — prominent shareable code
- Referral monthly count: 12px weight 400 `#757575` — "{N} referrals this month"
- Info page field labels: 12px weight 400 `#9E9E9E` — "Full Name", "Phone Number", "Date of Birth"
- Info page field values: 16px weight 400 `#212121` — read-only display
- Limits & Fees page section heading: 16px weight 700 `#212121`
- Maintenance modal title: 20px weight 700 `#212121` — "System Under Maintenance"
- Maintenance modal body: 12px weight 400 `#757575` — descriptor text (single exception: 12px instead of 16px for modal body secondary text, as per standard dialog pattern)
- Update modal title: 20px weight 700 `#212121` — "Software Update"
- Error boundary heading: 20px weight 700 `#212121` — "Something went wrong"
- Error boundary body: 16px weight 400 `#757575`

Source: Phase 1 UI-SPEC typography; Phase 6 UI-SPEC; Pencil SQ80R, XDpyV, Nl1Uy; UI-UX-REVIEW.md §1.2.
