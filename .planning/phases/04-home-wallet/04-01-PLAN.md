**3. Create `src/app/api/wallet/route.ts`** — GET endpoint:
- Import `createClient` from `@/lib/supabase/server`
- Get authenticated user via `supabase.auth.getUser()` — return 401 if no user
- Query `wallets` table: `select('id, balance, currency, max_topup').eq('user_id', user.id).single()`
- Query `user_profiles` table: `select('first_name, wallet_id').eq('id', user.id).single()`
- Return JSON: `{ wallet: { id, balance, currency, max_topup }, profile: { first_name, wallet_id } }`
- Balance returned as raw bigint satang — client formats via `formatCurrency`
- On missing wallet (no row), return `{ wallet: null, profile }` — dashboard shows zero state
- Return 500 with `{ error: 'Failed to fetch wallet' }` on DB errors

**4. Create `src/app/api/wallet/transactions/route.ts`** — GET endpoint:
- Same auth pattern as wallet route
- Query `transactions` table: `select('id, type, amount, currency, status, description, created_at').eq('user_id', user.id).order('created_at', { ascending: false }).limit(5)`
- Return JSON: `{ transactions: Transaction[] }` — amounts as raw bigint satang
- Empty array if no transactions (not an error)
- Return 500 with `{ error: 'Failed to fetch transactions' }` on DB errors

**5. Create `src/hooks/use-wallet.ts`** — client-side SWR hooks:
- `'use client'` directive at top
- Install SWR if not installed: check package.json first, run `npm install swr` if missing
- Create a simple fetcher: `const fetcher = (url: string) => fetch(url).then(r => { if (!r.ok) throw new Error('fetch failed'); return r.json() })`
- `useWallet()` hook: `useSWR('/api/wallet', fetcher, { revalidateOnFocus: true, dedupingInterval: 30000 })` — per D-10. Returns `{ data, error, isLoading, mutate }`.
- `useRecentTransactions()` hook: `useSWR('/api/wallet/transactions', fetcher, { revalidateOnFocus: true, dedupingInterval: 30000 })`. Returns `{ data, error, isLoading, mutate }`.
- Export both hooks and the `mutate` functions for external revalidation after transaction actions (per D-10).
