   - `wallets`: id (uuid PK), userId (uuid, notNull), balance (bigint mode:number, default 0), currency (text, default 'THB'), maxTopup (bigint mode:number, default 2500000), createdAt (timestamptz), updatedAt (timestamptz)
   - `transactions`: id (uuid PK), userId (uuid), type (text), amount (bigint mode:number), currency (text default 'THB'), convertedAmount (bigint mode:number, nullable), convertedCurrency (text nullable), exchangeRate (numeric(10,4) nullable), fee (bigint mode:number default 0), status (text default 'pending'), recipientId (uuid nullable), channel (text nullable), referenceNumber (text), description (text), metadata (text nullable), createdAt (timestamptz), updatedAt (timestamptz)
   - `userProfiles`: id (uuid PK), fullName (text), phone (text), countryCode (text default '+66'), walletId (text), kycStatus (text default 'not_started'), language (text default 'en'), avatarUrl (text nullable), firstName (text nullable), lastName (text nullable), dateOfBirth (text nullable), nationality (text nullable), idType (text nullable), idNumber (text nullable), idExpiry (text nullable), passcodeHash (text nullable), registrationComplete (boolean default false), registrationStep (integer default 1), passcodeAttempts (integer default 0), passcodeLockedAt (timestamptz nullable), webauthnCredentialId (text nullable), webauthnPublicKey (text nullable), webauthnCounter (bigint mode:number default 0), webauthnChallenge (text nullable), dailyLimitSatang (bigint mode:number default 5000000), monthlyLimitSatang (bigint mode:number default 20000000), createdAt (timestamptz), updatedAt (timestamptz)
   - `cards`: id (uuid PK), userId (uuid), cardNumberMasked (text), cardNumberEncrypted (text nullable), cvvEncrypted (text nullable), expiryMonth (integer), expiryYear (integer), balance (bigint mode:number default 0), isFrozen (boolean default false), status (text default 'active'), createdAt (timestamptz)
   - `recipients`: id (uuid PK), userId (uuid), fullName (text), phone (text), countryCode (text default '+95'), nrc (text nullable), occupation (text nullable), transferPurpose (text nullable), relationship (text nullable), address (text nullable), isFavorite (boolean default false), createdAt (timestamptz), updatedAt (timestamptz)

   CRITICAL: Every `bigint` column MUST use `{ mode: 'number' }` — existing TypeScript types use `number`, not JS `BigInt`.
   Use SQL column names as first arg: `uuid('user_id')`, `bigint('balance', { mode: 'number' })`, etc.
   JS property names are camelCase: `userId`, `balance`, `maxTopup`, etc.

3. Create `src/db/index.ts`:
   ```typescript
   import { neon } from '@neondatabase/serverless'
   import { drizzle } from 'drizzle-orm/neon-http'
   import * as schema from './schema'

   const sql = neon(process.env.DATABASE_URL!)
   export const db = drizzle({ client: sql, schema })
   ```

4. Create `drizzle.config.ts` at project root:
   ```typescript
   import { defineConfig } from 'drizzle-kit'

   export default defineConfig({
     out: './drizzle',
     schema: './src/db/schema.ts',
     dialect: 'postgresql',
     dbCredentials: {
       url: process.env.DATABASE_URL!,
     },
   })
   ```

5. Add to `.env.local.example`:
   ```
   # Database (Drizzle ORM — get from Supabase Dashboard > Settings > Database > Connection string > URI, Transaction pooler port 6543)
   DATABASE_URL=postgresql://postgres.[ref]:[password]@aws-0-[region].pooler.supabase.com:6543/postgres
   ```
