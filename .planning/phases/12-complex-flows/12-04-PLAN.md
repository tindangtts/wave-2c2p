    const [step, setStep] = useState<CaptureStep>('capture-front')

    Revised step union: `'capture-front' | 'review-front' | 'capture-back' | 'review-back' | 'confirm' | 'submitted'`

    After review-back "Use Photo" → setStep('confirm')

    **Confirm screen (step='confirm'):**
    - BackHeader: t('workPermitUpdate.title'), onBack → setStep('review-back')
    - Two thumbnail previews (front + back) in small rounded boxes
    - Yellow submit button: t('workPermitUpdate.submitCta'), disabled when isSubmitting
    - When isSubmitting: show t('workPermitUpdate.submittingLabel') instead

    **handleSubmit:**
    ```typescript
    const handleSubmit = async () => {
      setIsSubmitting(true)
      try {
        const res = await fetch('/api/mock-kyc/work-permit-update', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ document_type: 'work_permit' }),
        })
        const data = await res.json()
        setResult({ success: data.success, verificationId: data.verification_id, error: data.rejection_reasons?.[0] })
        setStep('submitted')
      } catch {
        setResult({ success: false, error: t('workPermitUpdate.failBody') })
        setStep('submitted')
      } finally {
        setIsSubmitting(false)
      }
    }
    ```

    **Submitted screen (step='submitted'):**
    - No BackHeader (terminal screen)
    - If result.success:
      - Icon (CheckCircle2 green, 64px), title=t('workPermitUpdate.successTitle'), body=t('workPermitUpdate.successBody')
      - CTA yellow button: t('workPermitUpdate.successCta') → router.push('/profile')
    - If !result.success:
      - Icon (XCircle red, 64px), title=t('workPermitUpdate.failTitle'), body=result.error
      - CTA outline button: t('workPermitUpdate.failCta') → setResult(null) + setStep('capture-front') (restart flow)

    **Back navigation / Cancel:**

    - During other capture steps: BackHeader onBack → previous step
    - Cancel dialog (AlertDialog controlled):

      - Confirm button: t('workPermitUpdate.cancelDialog.confirm') → router.push('/profile')
